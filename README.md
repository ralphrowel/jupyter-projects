Jupyter Projects Collection
Overview

This repository contains my collection of Jupyter Notebook projects, where I explore data analysis, visualization, and Python programming concepts.
Each notebook focuses on a different dataset or topic — applying what I’ve learned in Python, OOP, Pandas, NumPy, Matplotlib, and Seaborn.

These projects serve both as learning exercises and portfolio samples for data analytics and system development.

Tools & Libraries

* Jupyter Notebook – Interactive environment for running Python code

* Python 3

* Pandas – Data manipulation and analysis

* NumPy – Mathematical and statistical operations

* Matplotlib – Visualization and plotting

* Seaborn – Advanced data visualization
